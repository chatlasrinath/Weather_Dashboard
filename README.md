# Weather_Dashboard
The Weather Dashboard is a responsive web application that displays real-time weather information using live API data. The Weather Dashboard is a responsive web application that displays real-time weather information using live API data. Users can search for any city to view current weather conditions, hourly forecasts, and a 7-day forecast, all presented through an interactive and user-friendly interface.
